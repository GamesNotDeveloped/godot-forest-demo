# Godot Procedural Forest Demo

Demo of procedural terrain and foliage generation

## Addons

* Godot Uniplayer (https://github.com/GamesNotDeveloped/godot-uniplayer/)
* Godot Debug Menu (https://github.com/godot-extended-libraries/godot-debug-menu)
* Custom Biomes addon (initial implementation)

## Assets

### Models

All models are downloaded from sketchfab.com.
Textures and materials are slightly tuned for the project.
Check license.txt files for license information.

### Grass

Grass texture and shadres are taken from SimpleGrassTextured asset from IcterusGames (MIT licensed)
